twig
====

twig is an agent that enables people to get access to twitter freely via gtalk.

steps to get things work:

1. make a copy of config-example.py, fill it with your twitter app's consumer key and consumer secret.
2. rename it to config.py
3. make a copy of app-example.yaml, fill it with your google appengine id
4. rename it to app.yaml
5. upload it using google_appengine tools
6. now you can play with it!
